# 人脸识别

> 这是简单的通过虹软免费SDK做的一套人脸识别系统的Java实现

根据虹软实现的 人脸检测、追踪、识别 的JAVA解决方案

## 目录简介

----images 用于测试图片存储空间
----libs 项目包所需要的依赖
----src/main
    ----java/com/test java源码
         ----facecheck 人脸检测源码
         ----facematch 人脸检测与比对源码
         ----facetrack 人脸追踪源码
     ----resources 项目的配置文件,用于存放自己的sdk密匙设置
